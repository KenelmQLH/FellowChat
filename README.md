# FellowChat


## 项目简介
本项目旨在设计一个局域网工作群聊，并在群聊中加入项目管理功能

## 运行环境
（1）java : IntelliJ IDEA 2019
（2）javafx
（3）mysql

## 知识点
项目UI采用JAVAFX，后端主要用到socket编程、mySQL、文件读写等知识点

## 原理
项目原理主要包括C/S通信、P2P通信和FTP原理
